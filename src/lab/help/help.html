This step-wise manual will guide you through the process of installing and using this template.

So let's get started!!! Click on the Step 1 :installation menu.

Step 1: Installation

The installation process is very simple and involves the following steps.

Three easy steps

  1. 1. Download the zip file from this link template.zip
  2. 2. Unzip the file you just downloaded into a folder.
  3. 3. Open the index.html file in the browser.

Thats it! You now have the template running on your system.

Step 2: Directory structure

If you look into the directory where you just unzipped the template files, you will see the following file structure.

index.html This is the home page for your lab. It lists all of your experiments.
css/ All css files that affects global styling of the lab should be placed here.
css/common.css This file contains the basic style formatting for all the html elements.
css/style.css This file is the main style file describing the layout, typography and colors for all the html elements.
js/ All javascript files that affects the complete interactivity of the lab should be placed here.
js/jquery.js This is source jquery file needed for improved interactivity and ajax.
exp1/ This folder contains all the files related to an experiment 1.
exp1/index.html The experiment page that acts as a container for the content and the simulations of the experiment 1.
exp1/content.html This files describes the theory (content) behind the experiment 1.
exp1/experiment demo This is a simulation file (java applet/flash swf /etc) for the experiment 1. Its link should be included in content.html file.

Step 3: Configure

This template features a demo of a science lab with a sample "simple pendulum" experiment. To configure this template so that it features your lab and its experiments, follow the steps given below.

Adding/Modifing the content

Locate content.html file (It is situated in exp1/ folder) and open it in your favorite html editor. Enter the title of the experiment inside the h1 tag, you see at the top of the page. As you can see, this file is divided into number of sections (called divs in html). Each section has a unique id which is used to refer to its content globally (from other files). For example, in the sample content.html file, the id for the first div is "introduction" so that it points to introduction part of your experiment content. Change the given ids of all the sections of you want to the id-names that you like. Just remember ids should be unique and they should not contain spaces. Use" -" symbol to connect multiple words. For example, "new-simulations" is a valid id.

Inside the div, you have an h2 tag which is used to create the menu for the experiment page. Therefore, Whatever text you will enter here, will also appear on the main navigational menu on the experiment page (exp1/index.html).

Finally, we can write any text content after the h2 tag. You can also insert images with img tag.

Refer to w3school page for nice introduction to HTML tags. W3schools HTML reference

Add the experiment

First, replace the pendulum.swf file with your experiment. Then locate the section with an id as "experiments" inside the content.html file. Replace the embed code so that it points to your experiment. For flash based experiments it is easy, just replace the name of swf file with name of your experiment file. Refer to web tutorials on embedding java applets or flash objects in HTML pages.
